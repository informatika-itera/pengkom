Cakupan Materi UTS

Materi Komprehensif dari Pertemuan 1 sampai 6.

  1. Dasar: Computational Thinking, Variabel, Tipe Data.
  2. Logika: Percabangan (If-Elif-Else), Operator Logika.
  3. Looping: For, While, Nested Loop.
  1. Data Structure: List operations.
  2. Algorithm: Problem solving sederhana.

Kisi-Kisi CBT: Pilihan Ganda

Pilihan Ganda (35 Soal)

  • Konsep dasar programming (6 soal)
  • Variabel dan tipe data (6 soal)
  • Operator aritmatika & logika (6 soal)
  • Percabangan (If-Else) (6 soal)
  • Perulangan (For/While) (6 soal)
  • List dan operasinya (5 soal)

Kisi-Kisi CBT: Benar/Salah

Benar/Salah (15 Soal)

  • Syntax dasar Python (5 soal)
  • Konsep variabel dan tipe data (3 soal)
  • Operator dan ekspresi (3 soal)
  • Struktur kontrol (If-Else) (2 soal)
  • Loop dan iterasi (2 soal)

Kisi-Kisi CBT: Isian Singkat

Isian Singkat (10 Soal)

  • Output program Python sederhana (4 soal)
  • Syntax dan struktur kode (3 soal)
  • Debugging error sederhana (3 soal)

Contoh Soal Isian Singkat: Output Program

Soal 1

Apa output dari kode berikut?

x = 5
y = 3
print(x + y)

Jawaban: 8

Soal 2

Apa output dari kode berikut?

numbers = [1, 2, 3, 4, 5]
print(numbers[2])

Jawaban: 3

Contoh Soal Isian Singkat: Syntax & Struktur Kode

Soal 1

Lengkapi struktur if-else yang benar:

___ x > 10:
    print("Besar")
___:
    print("Kecil")

Jawaban: if, else

Soal 2

Lengkapi struktur for loop yang benar:

___ i ___ range(3):
    print(i)

Jawaban: for, in

Contoh Soal Isian Singkat: Debugging Error

Soal 1

Perbaiki syntax error pada kode berikut:

if x > 5
    print("Besar")

Jawaban: if x > 5:

Keterangan: Seharusnya ada tanda titik dua (:) setelah kondisi if
