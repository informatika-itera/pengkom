Apa itu Library?

Library (atau Modul) adalah kumpulan kode/fungsi yang sudah ditulis orang lain untuk kita gunakan.

"Jangan menemukan kembali roda (Don't reinvent the wheel)."


# Cara Menggunakan
import nama_library

# Atau
import nama_library as alias

# Atau
from nama_library import fungsi_tertentu

Library: Math - Fungsi Dasar

Fungsi matematika tingkat lanjut.

Fungsi Dasar


import math

print(math.sqrt(25))    
# 5.0 (akar kuadrat)
print(math.pow(2, 3))   
# 8.0 (pangkat)
print(math.pi)          
# 3.1415... (nilai pi)

Pembulatan


import math
# jika sudah import math 
# tidak perlu diulang

print(math.ceil(4.2))   
# 5 (pembulatan ke atas)
print(math.floor(4.8))  
# 4 (pembulatan ke bawah)

Library: Math - Trigonometri

Fungsi trigonometri dengan input dalam radian.

Trigonometri


# Input dalam Radian
print(math.sin(math.pi/2)) 
# 1.0
print(math.cos(0))         
# 1.0

Konversi Sudut


sudut = math.radians(90)
print(math.sin(sudut))     
# 1.0

sudut = math.degrees(math.pi/2)
print(sudut)               
# 90.0

Library: Random

Untuk membuat simulasi atau pengacakan.


import random

# Angka acak 0.0 - 1.0
print(random.random())

# Angka bulat acak (dadu)
print(random.randint(1, 6))

# Pilihan acak dari list
buah = ["Apel", "Jeruk", "Mangga"]
print(random.choice(buah))

Gunakan random.seed() jika ingin hasil acak yang konsisten (reproducible).

Intro to Pandas

Library paling populer untuk Analisis Data.

Kode Python


import pandas as pd

# Membuat DataFrame (Tabel)
data = {
    'Nama': ['Budi', 'Siti', 'Andi'],
    'Usia': [20, 19, 21],
    'Kota': ['Lampung', 'Jakarta', 'Bandung']
}

df = pd.DataFrame(data)
print(df)

Output Terminal

   Nama  Usia     Kota
0  Budi    20  Lampung
1  Siti    19  Jakarta
2  Andi    21  Bandung

Pandas - DataFrame

Struktur data tabular seperti Excel.

DataFrame adalah struktur data tabular dengan baris dan kolom untuk analisis data.

Ciri-ciri:

  • Index otomatis dari 0
  • Kolom sesuai dengan nama yang diberikan
  • Bisa berbagai tipe data

Keunggulan:

  • Mudah dibaca manusia
  • Powerful untuk manipulasi data
  • Banyak method built-in
