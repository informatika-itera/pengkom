Apa itu Fungsi?

Fungsi adalah blok kode yang diberi nama dan dapat dipanggil berulang kali.

Fungsi membantu kita:

  • Menghemat kode (Don't Repeat Yourself / DRY)
  • Mengorganisir kode agar rapi

Keuntungan Fungsi


Reusable

Kode dapat dipanggil berkali-kali tanpa perlu menulis ulang


Modular

Program terbagi menjadi bagian-bagian kecil yang terorganisir


Maintainable

Mudah diperbaiki dan dikembangkan


Shareable

Dapat digunakan di program lain

Cara Mendefinisikan Fungsi

Syntax Dasar


def nama_fungsi(parameter):
    # Blok kode perintah
    # ...
    return hasil  # Opsional
  • def = keyword untuk mendefinisikan fungsi
  • nama_fungsi = identifier unik
  • parameter = input (opsional)
  • return = output (opsional)

Langkah-langkah

  1. Tulis keyword def
  2. Beri nama fungsi
  3. Tentukan parameter (jika ada)
  4. Tulis blok kode dengan indentasi
  5. Gunakan return jika perlu

Contoh Fungsi Sederhana

Fungsi Tanpa Parameter


def sapa_dunia():
    print("Halo Dunia!")

# Memanggil fungsi
sapa_dunia()
# Output: Halo Dunia!

Fungsi ini tidak menerima input dan tidak mengembalikan nilai.

Fungsi Dengan Return


def kuadrat(x):
    return x * x

# Memanggil dan menggunakan hasil
hasil = kuadrat(5)
print(hasil)  # Output: 25

Fungsi ini menerima parameter dan mengembalikan nilai.

Parameter & Argumen

Parameter adalah variabel yang didefinisikan dalam fungsi untuk menerima data.

Satu parameter


def sapa_nama(nama):
    print(f"Halo, {nama}!")

sapa_nama("Budi")   # Halo, Budi!
sapa_nama("Siti")   # Halo, Siti!

Dua parameter


def tambah(a, b):
    print(f"Hasil: {a + b}")

tambah(5, 3)  # Hasil: 8

Catatan: Gunakan f"teks {variabel}" untuk menyisipkan variabel ke string.

Return Value

print() hanya menampilkan ke layar.

return mengembalikan nilai ke pemanggil, sehingga bisa disimpan dalam variabel atau diolah lagi.


def kuadrat(x):
    return x * x

hasil = kuadrat(5)
print(hasil + 10)  
# Output: 35 (25 + 10)

Scope Variabel (Cakupan)

Global Scope

Dibuat di luar fungsi. Bisa diakses di mana saja.


x = 10  # Global

def cek():
    print(x) # Bisa!

Local Scope

Dibuat di dalam fungsi. HANYA hidup di dalam fungsi.


def hitung():
    y = 5  # Local
    print(y)

hitung() # tercetak 5
print(y) # tapi ini ERROR!
