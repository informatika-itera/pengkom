Apa itu List?

List adalah struktur data yang dapat menyimpan banyak nilai dalam satu variabel.

Bayangkan sebuah Rak Buku atau Loker yang bernomor.

Ilustrasi List

A
B
C
D
E

[4]

Membuat List


# List Kosong
kosong = []

# List String
buah = ["apel", "jeruk", "mangga"]

# List Campuran
campur = [1, "dua", 3.0, True]

# List dari Range
angka = list(range(1, 6))  # [1, 2, 3, 4, 5]

Mengakses Elemen

Index Positif & Negatif


buah = ["apel", "jeruk", 
        "mangga", "pisang"]

print(buah[0])   # apel
print(buah[2])   # mangga
print(buah[-1])  # pisang
Output Terminal
apel
mangga
pisang

SLICING (Potongan)


# [start:stop]
buah[0:2]
# ["apel", "jeruk"]

buah[1:]
# ["jeruk", "mangga", "pisang"]
Output Terminal
['apel', 'jeruk']
['jeruk', 'mangga', 'pisang']

Mengubah & Menambah Data

Mengubah Elemen

Gunakan index untuk mengubah nilai


buah = ["apel", "jeruk"]
buah[0] = "anggur"
print(buah)
Output Terminal
['anggur', 'jeruk']

Menambah Elemen

Gunakan append() untuk menambah di akhir


buah = ["apel", "jeruk"]
buah.append("mangga")
print(buah)
Output Terminal
['apel', 'jeruk', 'mangga']

Menyisipkan & Menghapus Data

Menyisipkan Elemen

Gunakan insert(index, nilai)


buah = ["apel", "mangga"]
buah.insert(1, "jeruk")
print(buah)
Output Terminal
['apel', 'jeruk', 'mangga']

Menghapus Elemen

Gunakan remove(nilai)


buah = ["apel", "jeruk", "mangga"]
buah.remove("jeruk")
print(buah)
Output Terminal
['apel', 'mangga']

Method List Populer

Method adalah fungsi yang dimiliki oleh objek list untuk memanipulasi datanya. Berikut adalah method-method populer:

Method Fungsi
append(x) Tambah x di akhir
insert(i, x) Sisipkan x di index i
pop() Hapus & ambil elemen terakhir
remove(x) Hapus x pertama yang ditemukan
sort() Urutkan list (ascending)
reverse() Balik urutan list
count(x) Hitung jumlah kemunculan x

Fungsi Built-in

len(L) Panjang list
max(L) Nilai terbesar
min(L) Nilai terkecil
sum(L) Total jumlah 

angka = [5, 2, 8, 1, 9]

print(len(angka))  # 5
print(max(angka))  # 9
print(min(angka))  # 1
print(sum(angka))  # 25

Iterasi List (Looping)

Iterasi memungkinkan kita mengakses setiap elemen dalam list satu per satu. Ada beberapa cara untuk melakukannya:


buah = ["apel", "jeruk", "mangga"]

# Cara 1: Langsung (Recommended)
for item in buah:
    print(item)

# Cara 2: Pakai Index
for i in range(len(buah)):
    print(f"Index {i}: {buah[i]}")

# Cara 3: Enumerate (Index + Item)
for i, item in enumerate(buah):
    print(f"{i}: {item}")
